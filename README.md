# FeelGoodFeatures

A project to help me learn Go (Golang) :-)

This will use be an api which uses a local LLM (probably Ollama) to scrape a pre-configured list of websites for 'feel good' news stories.

A list of story summaries will then be returned to the client in JSON.

## Installation

To run this locally - 

```bash
go run .
```

To build and run it for shipping it  - 

```bash
go build -o feelgoodfeatures
./feelgoodfeatures
```
